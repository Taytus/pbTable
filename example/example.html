JQuery quick filter table plug-in

The easiest way to very elegant filter table data.

Examples Download

Version: 0.2 Beta
To install: call JQuery and javascript plugin 

<script src="http://ajax.googleapis.com/ajax/libs/jquery/2.0.3/jquery.min.js"></script>
<script src="quickFilterTable.js"></script>

Example 2: using your custom search box input

Try to type the listed fuits or the producer country (hidden attribute) like "Argentina", "Brasil", "Chile", "Ecuador" or "Panama"

Fruits Color Productor
Apple Green Argentina
Banana Yellow Ecuador
Cherry Red Panama
Kiwi Green Chile
Strawberry Red Brasil
Melon Red Brasil 
Just add your input search somewhere into "<body>" tag. Add the "search-in" attribute specifing the table id.

<input type="text" search-in="fruits-table">
<table id="fruits-table">
	<thead>
		<th>Column 1</th>
		<th hidden>Column 2</th>
	</thead>
	<tbody>
		<td>Value 1</td>
		<td hidden>Hidden value </td>
	<tbody>
</table>

Just add your input search somewhere into "<body>" tag. Add the "search-in" attribute specifing the table id.
Add a button or an "a" tag with the "quickTag" class. You have to add the "input-search" to indicate the search box id.


<input id="colourSearchBox" type="text" class="form-control" search-in="colours-table">
<button class="quickTag" input-search="colourSearchBox">Primary<button>
<button class="quickTag" input-search="colourSearchBox">Secondary<button>
<table id="fruits-table">
	<thead>
		<th>Column 1</th>
		<th hidden>Column 2</th>
	</thead>
	<tbody>
		<td>Value 1</td>
		<td hidden>Hidden value </td>
	<tbody>
</table>
